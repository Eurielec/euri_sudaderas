# Eurielc's store

Frontend and backend for the Eurielec's store

## Setup

To see it live:

- Clone the repository.
- Start the Strapi server: `cd ./tienda-backend && yarn && strapi dev`.
- In an other tab, start the nextjs server: `cd ./nextjs-frontend && yarn && yarn dev`.
